# react-todo-app

Test app built with [vite](https://vitejs.dev/) using React + Typescript.

# Prerequisites
- [yarn](https://yarnpkg.com/)

# How to run
- Run `yarn install`
- Run `yarn dev`

Project will be running on http://localhost:5173/
